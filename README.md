# BNMAP v1.2 
> Assistência modular para análise de redes.

## 🛠️ Como Instalar (Qualquer Terminal)

### No Termux:
pkg install git nmap bc -y
git clone https://github.com/mrmaestrox156-debug/BNMAP.git
cd BNMAP && chmod +x * && ./BNMAP.sh

### No Kali Linux / Debian / Ubuntu:
sudo apt update && sudo apt install git nmap bc -y
git clone https://github.com/mrmaestrox156-debug/BNMAP.git
cd BNMAP && chmod +x * && ./BNMAP.sh
